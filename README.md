# 🐍 网页版贪吃蛇

这是一个纯前端实现的经典贪吃蛇小游戏，单文件 HTML 即可运行，支持 GitHub Pages 一键部署，手机电脑都能玩。

---

## 🎮 游戏玩法
- **电脑端**：方向键 / WASD 控制蛇的移动
- **手机端**：触屏滑动控制方向
- 吃到食物得分，撞墙或撞到自己游戏结束
- 支持本地存储最高分记录

## ✨ 项目特点
- 纯 HTML + CSS + JavaScript 实现，无任何依赖
- 深色主题界面，适配 PC 和移动端
- 支持开始、暂停、重新开始功能
- 可通过 GitHub Pages 一键部署为公开网页
